Overview

A parameterized array type that casts each element using an ActiveModel type. Instances are created per-attribute since each may have a different element type.

Casting does not raise on invalid elementsâ€”errors are deferred to the validation callback in Base.

Examples:

typed = TypedArray.new(element_type: :integer)
typed.cast(["1", "2", "3"]) # => [1, 2, 3]

def cast(value)
  return nil if value.nil?
  return value unless value.is_a?(Array)

  value.map { |element| @element_caster.cast(element) }
end
