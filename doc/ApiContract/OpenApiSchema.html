Overview

Generates OpenAPI 3.0 / JSON Schema compatible schema hashes from contract class definitions. Each contract can produce a schema that maps its typed attributes to JSON Schema properties.

Examples:

UserContract.open_api_schema
# => { "type" => "object", "properties" => { ... }, "required" => [...] }
