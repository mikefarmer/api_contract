Overview

Descriptor for polymorphic nested contracts. Returned by the one_of class method and used by the contract: attribute option.

During instantiation, each candidate contract is tried in declaration order. The first candidate whose schema_validate! does not raise is used. If no candidate matches, behavior depends on the permissive flag.

Examples:

attribute :address, contract: one_of('USAddress', 'UKAddress')

def resolve(hash, resolver:)
