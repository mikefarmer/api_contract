Overview

Provides the normalizes DSL for transforming attribute values during initialization. Normalizers run after type coercion but before the contract is frozen.

Included automatically by Base.

Examples:

class UserContract < ApiContract::Base
  attribute :email, :string
  normalizes :email, with: ->(email) { email.strip.downcase }
end
